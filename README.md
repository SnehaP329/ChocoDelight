# ChocoDelight

A user-friendly chocolate shopping application built using Android Studio and Firebase. This app allows users to browse an extensive catalog of delicious chocolates, view product details, and manage their shopping cart,stores the orders.

## Features
* **User Authentication:** Secure user sign-up and login powered by Firebase.
* **Product Catalog:** Browse a dynamically managed selection of chocolate products.
* **Shopping Cart:** Add, update, and manage items seamlessly before checkout.
* **Real-time Updates:** Smooth backend integration using Firebase services.

## Tech Stack
* **Frontend:** Kotlin, XML (Android Studio)
* **Backend & Database:** Firebase Authentication, Cloud Firestore
